# Car Game

## Description
Car Dodge Game is a browser-based game where the player controls a car to avoid collisions with enemy cars. The game becomes progressively challenging as the speed of enemy cars increases. The game tracks the current score and high score.

## Features
- **Player-controlled car movement**: Use the left and right arrow keys or the mouse cursor to move the car.
- **Enemy cars**: Randomly spawn and move down the screen.
- **Score tracking**: Displays the current score and the highest score achieved.
- **Game over**: Displays a message and provides a restart button when the player collides with an enemy car.

## How to Run
1. Clone the repository and navigate to the project directory:
   ```bash
   git clone <repository-url>
   cd carGame
   ```
2. Start the application using the `serve` command:
   ```bash
   serve .
   ```
3. Open your browser and go to the URL shown in the terminal after running `serve`
